# ⌨️ 常用指令

## 基础指令

* `/cd`打开菜单
* `/rt`随机传送（即刻传输但有冷却）
* `/spawn`回到主城
* `/tpa`传送到另一玩家处
* `/tpahere`请求玩家传送到你这
* `/warp`传送至服务器传送点
* `/home`设置「家」传送点（非 Pro 上限为5）
* `/shop`打开系统商店
* `/msg`私信玩家
* `/pay`转款给其他玩家
* `/kit`领取礼包
* `/c`打开伙伴面板
* `/tags`佩戴已有称号
* `/list`在线者一览
* `/baltop` 马桶币排行榜

## 更多指令

服务器更多指令请参阅[插件说明](../guidebook/plugins/)页面。
